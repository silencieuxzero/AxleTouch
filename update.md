# AxleTouch 更新日志

## V1.0.0 — 2026-07-14

### 新增功能

- 识图厂商新增**阶跃星辰**（`stepfun`，模型 `step-1.5v-mini`）多模态选项，支持图像识别、文本理解及跨模态内容分析

### 界面优化

- 统一色彩体系，新增 15 个色彩常量，替换所有硬编码颜色值
- ContentBar 气泡尺寸与质感提升：高度 68px、宽度 130px、阴影 5 层、圆角 14px
- ContentBar 进度条高度增至 3px，更醒目
- 动画时长微调：显示 350ms、隐藏 300ms，交互更敏捷
- ContentBar 支持自适应高度，长文本自动换行并动态调整气泡高度
- InputPopup 输入框高度随桌宠图标尺寸联动调整
- 文本输出增加 `line-height: 1.4`，提升多行排版可读性

### 交互优化

- 桌宠图标默认尺寸 100 → 135（面积增大约 82%）
- 配置文件中 `icon_size` 默认值同步更新，且运行时强制执行最小尺寸兜底
- 隐藏后 ContentBar 自动重置为默认 68×130，避免下次显示尺寸错乱

### 性能优化

- `_card_rect()` 缓存结果，仅在窗口尺寸变化时重建，减少 paintEvent 开销
- 新增模块级 `_NOW()` 统一时间戳格式化，减少重复 import 与方法查找
- `save_config()` 直接读取文件 JSON 后合并，减少一次配置迁移调用链
- 打印语句统一替换为 f-string，减少临时字符串拼接开销

### 代码质量

- 删除 `config_manager.py` 中 3 个未使用常量
- 删除 `widgets.py` 中 1 个未使用变量
- `tts_manager.py` 删除 `_QwenTTSWorker` 与 `_BailianTTSWorker` 两个冗余类，精简 248 行重复代码
- 新增阶跃星辰识图厂商的配置迁移支持，旧配置自动升级
