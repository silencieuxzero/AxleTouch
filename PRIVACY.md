# 数据收集与隐私声明

**最后更新日期：2026年7月11日**  
**适用软件：雨竹桌宠（开源 GPL v3）**

## 1. 概述
本软件是一个基于视觉语言模型（VLM）以及大语言模型（LLM）的桌面宠物程序。您在使用时，某些设备数据会被发送给第三方人工智能服务提供商，以便生成对话回复、识别图片和理解屏幕内容。  
**本软件的开发者不会收集、存储或访问您的任何个人数据。** 但您必须了解软件在您设备上的具体行为。

## 2. 我们收集什么数据
### 2.1 自动收集的数据
- **当前活动窗口的标题**  
  为实现“自动轮询”功能，软件会在后台定期读取您当前窗口的标题文字（例如浏览器标签页标题、文档名称）。该过程静默进行，无需您额外操作。

### 2.2 您主动提供的数据
- **对话文本**：您在对话框中输入的任何文字。
- **图片**：您通过“向雨竹发送图片”功能选择的图像文件。
- **屏幕内容**：您使用“邀请雨竹查看屏幕”功能时，软件截取的当前屏幕画面。
- **自定义提示词与其他设置**：您配置的个性化指令。

## 3. 这些数据如何被使用
所有上述数据均通过加密网络连接（QNetwork）直接发送至您所选择的**API 服务商**，仅用于以下目的：
- 生成“雨竹”的对话回复
- 理解图片内容并给出相关回应
- 根据屏幕内容提供上下文相关的互动
- 执行您设置的自定义提示词逻辑

**本软件开发者不会将数据用于其他任何目的，亦不进行数据分析、用户画像或广告投放。**

## 4. 数据共享与第三方
您的数据会被发送至第三方 API 供应商。您使用的具体供应商取决于您配置软件时填入的 API 端点及密钥。  
**请您务必自行查阅该供应商的隐私政策**，了解他们如何处理您的数据。常见供应商包括（但不限于）：
- StepFun （阶跃星辰 API）
- OpenAI (ChatGPT API)
- Google (Gemini API)
- Anthropic (Claude API)
- 阿里云（通义千问 API）
- 百度（文心一言 API）

开发者不对这些第三方的隐私保护措施承担责任。

## 5. 数据存储与安全
- **本地存储**：软件可能将您的自定义设置（如API密钥、提示词）存储在本地配置文件中，不会上传给开发者。
- **传输安全**：数据通过 HTTPS/TLS 加密传输至 API 服务商。
- **开发者侧**：开发者**不拥有任何服务器，不收集、不存储、不看到**您的窗口标题、对话、图片和屏幕截图。

## 6. 您的控制权
- 图片和屏幕查看功能**完全由您手动触发**，不主动操作则不会传输此类数据。
- 卸载软件即可彻底移除所有本地痕迹。
